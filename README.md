# iStudy

这是一款类似**我要当学霸**的学习监督功能的App,提供以倒计时形式禁止非系统App的使用，帮助你提高自觉性。同时，结合娱乐性的桌面宠物作为你的学习监督小管家，以便你在无聊时，在忍不住想玩手机时，可以让你解解馋，逗一逗小宠物。
